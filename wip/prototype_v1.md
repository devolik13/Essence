## Формулы (актуальные)

```
Макс HP   = 50 + Здоровье × 5
Макс Мана = 20 + Мана × 3
Реген HP  = Макс HP / 60  в сек
Реген Маны= Макс Мана / 30 в сек

Физзащита = Стойкость / (Стойкость + 125)   → макс 80%
Магзащита = Воля / (Воля + 125)              → макс 80%

Урон (ближний): BaseDamage_оружия × (1 + Сила / 100)
Урон (дальний): BaseDamage_оружия × (1 + Ловкость / 100)
Урон (магия):   BaseDamage_заклинания × (1 + Интеллект / 100)

Попадание = Точность / (Точность + Уклонение)   [физика/дальний]
Магия всегда попадает.

Крит шанс = Удача / (Удача + 500)   → при кап 500 = 50% крит
Крит = урон × 1.5

XP стоимость: текущий_стат × 10 за +1
Ранг = сумма 10 статов / 10

Кап статов (предварительно): 500 по каждому статy
```

## AI поведение мобов

- **Ближний бой:** агро → chase → attack при dist < 44px
- **Дальний бой:** агро → chase → attack при dist < range×0.85, отступает если игрок ближе range×0.42
- **Пассивные:** не агрируют, убегают (wanderAI)
- **Поводок:** при dist > LEASH_RANGE возвращается на спавн с полным восстановлением HP

## Реализованные системы

### Сфера
- 10 статов, XP-трекер сохраняется между телами
- Изученные заклинания хранятся навсегда
- Прогресс заклинаний по spellId
- Автосохранение в `localStorage` (ключ `essence_sphere_v1`)

### Тела и захват
- 3 стартовых тела на камне возрождения
- Захват мёртвых существ (E, каст 1.5с)
- Тело respawn: 15с в очереди → спавн нового
- Мёртвое тело: 30с до авто-исчезновения

### Бой
- 3 типа урона с разными формулами и статами
- Попадание/уклонение/крит
- Система AoE заклинаний с режимом прицеливания мышью
- Время каста блокирует движение
- Кулдаун отображается оверлеем + числом секунд на слоте

### UI
- Левая колонка: статы Сферы с XP-баром и капом, прогресс заклинания, квесты
- Правая колонка: инфо о теле, панель цели (клик на моба)
- Нижний центр: HP/мана тела, 8 слотов умений
- Полоска захвата / каста заклинания
- Лог событий (6 строк)
