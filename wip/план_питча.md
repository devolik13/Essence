# План питча — дорожная карта до показа издателю

> Рабочий план. Идём по этапам сверху вниз; внутри этапа — по порядку.
> Темп хобби: без дат, но с жёсткими критериями готовности (DoD) на каждый этап.
> Правило: не начинаем следующий этап, пока DoD текущего не закрыт.

## Цель

Не «продать идею» (идеи не покупают), а собрать три вещи, которые покупают:
1. **Играбельный вертикальный срез** (30–60 мин), доказывающий уникальную петлю.
2. **Материалы питча** (трейлер, дек, веб-ссылка на билд).
3. **Первичный трекшен** (страница, клипы, отклик аудитории).

**Главный аргумент питча** — петля двух миров: *«учись в чужом мире,
вселяясь в существ → защищай свой мир тем, что выучил»* + конфликтные
квесты со скрытыми игроками среди NPC (PvP, которого ни у кого нет).

## Этап 0 — фундамент (УЖЕ ЕСТЬ ✅)

- Деревня + 4 элементальные зоны, ~250 мобов, крафт (108 предметов), 87 умений
- Квесты тел (33), основная цепочка (12), диалоги NPC, пролог (16 страниц)
- **Полная EN/RU локализация** — питч сразу международный
- Сохранения (3 слота), редактор карт, оптимизации, настройки
- Phaser = веб-билд: издатель играет по ссылке за 10 секунд

## Этап 1 — Вертикальный срез (геймплейное ядро)

Состав среза: **пролог → создание персонажа → стартовая зона (вселения,
2–3 квеста тел) → босс Игнис → данж «Защита лаборатории» → титры/CTA.**

### 1.1 Данж «Защита лаборатории» ⟵ НАЧИНАЕМ ЗАВТРА
Стимпанк-арена. Твари Пустоты врываются уничтожить Машину Переноса.
- [ ] Родной персонаж (создаётся в начале игры; пока можно: генерируется из пролога)
- [ ] Арена-лаборатория: тайлсет/декор, Машина Переноса как защищаемый объект с HP
- [ ] 2–3 типа тварей Пустоты с механикой полуматериальности
      (обычный урон ослаблен, резонансные умения — полный)
- [ ] Резонаторное снаряжение: маппинг выученных школ на стимпанк-предметы
      (резонаторные перчатки = огонь и т.п.)
- [ ] Волны + финал: закрытие разрыва Печатью/частотами
- [ ] 1–2 близких NPC: диалог до и после (личная ставка)
- [ ] Нарративный мостик: после Игниса — тревога из лаборатории → переход

**DoD 1.1:** данж проходится от входа до закрытия разрыва без багов;
поражение (Машина уничтожена) обрабатывается; работает на обоих языках.

### 1.2 Игнис — настоящий босс
Сейчас боссы = обычные мобы (известная задача).
- [ ] HP-бар босса, 2–3 фазы (порог HP → смена паттерна)
- [ ] Использует T1/T2/T3 огня по фазам, арена-механика (зоны огня)
- [ ] Награда: Огненный шар + 1-я частота (уже есть) + переход к 1.1

**DoD 1.2:** бой длится 1.5–3 мин, читаемые фазы, нельзя зафейспалмить
с первого раза и нельзя заскучать.

### 1.3 Первые 15 минут (онбординг)
- [ ] Пролог → первое вселение в зверя максимально быстро
- [ ] Подсказки ключевых действий (вселение, выход, квест тела, скилл-бар)
- [ ] Именованные стартовые тела (Конрад/Рен/Эйла из лора — вместо generic)
- [ ] Срез маршрута: убрать всё, что уводит в сторону от пути к Игнису

**DoD 1.3:** свежий человек без подсказок автора доходит до первого
вселения ≤ 5 мин и понимает «фишку» игры.

### 1.4 Фирменный приём — смена тела в бою
- [ ] VFX/SFX/слоумо-момент захвата трупа врага в бою — должно ощущаться «вау»
- [ ] Это кадр №1 трейлера — полируем до блеска

**DoD этапа 1:** сквозное прохождение пролог→лаборатория за 30–60 мин,
без вылетов, на EN и RU.

## Этап 2 — Полировка витрины

### Арт-политика питча (РЕШЕНО)
Цель — играбельный процесс, не графика. Издатели смотрят «сквозь»
плейсхолдеры, но НЕ сквозь стилевой разнобой.
- **CraftPix/ассет-паки для окружения — ок** (лицензия позволяет, практика
  индустрии). В деке честно: «прототип-арт, финальный арт-дирекшн — с командой».
- **Главный риск — конфликт стилей**, не дешевизна. Склейка: единый
  тинт/палитра/освещение поверх всех ассетов (биомные тинты уже есть).
- **Кастом — только в 4 точки уникальности** (этого нет в паках, это IP):
  1) Сфера, 2) VFX захвата тела, 3) твари Пустоты, 4) Машина Переноса.
- Трейлер продают VFX/звук/монтаж («соки»), не качество спрайтов.

- [ ] Единый арт-стиль: ревизия ассетов на стилевые конфликты (одна стилистика)
- [ ] Анимированные спрайты ключевых существ среза (волк, медведь, Игнис, твари)
- [ ] **Фоновая музыка**: минимум 3 трека (деревня / бой+босс / лаборатория) —
      без музыки трейлер не собрать
- [ ] Баланс первого часа (кривая сложности до Игниса)
- [ ] Экран смерти/поражения, титры среза с контактами

**DoD этапа 2:** запись геймплея смотрится как «настоящая игра», не прототип.

## Этап 3 — Материалы питча

- [ ] **Веб-билд**: itch.io (приватная ссылка для издателей + публичная демка)
- [ ] **Трейлер 90–120 сек.** Обязательные кадры:
      1) вселение в волка и охота его глазами, 2) смена тела через труп в бою,
      3) Игнис, 4) разрыв в лаборатории и твари, 5) закрытие портала.
      Структура: хук (5 сек) → петля → уникальность → CTA
- [ ] **Пичдек 10–15 слайдов**: хук-фраза → фантазия игрока → петля двух миров →
      уникальные механики (вселение, конфликтные квесты со скрытыми игроками) →
      рынок и компы → роадмап (4 главы + 8 DLC-школ) → текущий статус/демо →
      команда → что просим (ask)
- [ ] **Одностраничник** (PDF: картинка, хук, петля, ссылка на билд, контакты)

**DoD этапа 3:** человек, впервые видящий проект, за 3 минуты (трейлер +
одностраничник) пересказывает фишку игры своими словами.

---

## Этап 4 — Публичность и трекшен (параллельно этапу 3)

- [ ] Страница на itch.io с демкой
- [ ] 3 коротких клипа для соцсетей/Reddit: захват тела в бою; квест глазами
      зверя; защита лаборатории. Смотрим, что резонирует
- [ ] Devlog (1 пост в 2–4 недели): механика вселения, «никто не знает кто
      из NPC игрок», лор резонанса
- [ ] Если отклик есть → Steam-страница и сбор вишлистов (вишлисты = главный
      аргумент на переговорах)

**DoD этапа 4:** есть цифры, которые можно показать (игры демки, охваты
клипов, вишлисты) — любые, но настоящие.

---

## Этап 5 — Рассылка и переговоры

- [ ] Список 20–30 издателей/фондов под жанр (инди-RPG, мультиплеер-lite)
- [ ] Персонализированные письма: 3 строки + одностраничник + ссылки
      (билд, трейлер). Без NDA-паранойи — ров это исполнение, не секретность
- [ ] Питч-сессии и фестивали (онлайн-ивенты издателей, шоукейсы)
- [ ] Честная позиция по команде: автор = гейм-дизайнер/визионер, ищется
      команда реализации (это нормальная заявка для co-dev/инкубаторов)

## Порядок работ — СОГЛАСОВАН (идём сверху вниз)

```
✅ 1.1 Данж лаборатории — ГОТОВ (волны → Колосс → ритуал → победа/поражение,
       арт игрока интегрирован, вход после Игниса через квест разрыва)
✅ 1.2 Босс Игнис — фазы/HP-бар/огненные зоны готовы (баланс — на плейтесте)

→ 1. ВАУ-захват тела (1.4): слоумо + вытяжка души + вспышка + звук  ⟵ ДЕЛАЕМ
  2. Финал среза: экран «Глава завершена» + титры/CTA + экран поражения
  3. Онбординг (1.3): подсказки [E]/квест тела/скилл-бар, срез маршрута
  4. Музыка: 3 трека (деревня / бой+босс / лаборатория) — файлы ищет Денис,
     систему зональной музыки подключает Claude
  5. Плейтест: полный прогон EN + RU руками (Денис), баланс Игниса 1.5–3 мин
✅ 6. Веб-билд: собран и проверен headless (Pitch Demo/лаборатория/музыка
       работают в prod). Инструкция по заливке: pitch/itch_upload.md —
       заливает Денис (аккаунт itch.io)
  7. Трейлер 90 сек + дек 10–15 слайдов (по готовым outline/shotlist)
```

Отложено сознательно: резонаторное снаряжение (лор-вишенка — в дек, не в демо).

## Что сознательно НЕ делаем до питча

- Остальные 3 зоны как боссы-механики (Акварис/Терра/Аэрос — после)
- 5 школ Главы 2 и 8 DLC-школ
- Сетевой код / MMO-часть (в питче — как roadmap, не как демо)
- Конфликтные квесты в мультиплеере (в деке — концепт-видео/слайд)
