# Лаборатория — план комнаты и промпты декора

> Арена 60×46 тайлов (1920×1472 px). Разрыв на севере, Машина в центре,
> игрок просыпается у Машины. Декор не должен мешать бою в центре.

## План комнаты (вид сверху)

```
┌────────────────────── СТЕНА (латунь/трубы) ──────────────────────┐
│  [шкаф][шкаф]      ░░░ РАЗРЫВ (зона тварей) ░░░      [генератор] │
│  [доска]                  — пусто —                  [полки]     │
│                                                                  │
│  [стол лаб. │ колбы]                            [верстак механика]│
│  [стул]                                         [ящики][бочка]   │
│                    ┌──────────────┐                              │
│  [книжный шкаф]    │  МАШИНА      │   [консоль с рычагами]       │
│  [письм. стол]     │  ПЕРЕНОСА    │   [катушка Теслы]            │
│  [кресло]          └──────────────┘                              │
│                 (точка пробуждения игрока)                       │
│                                                                  │
│  [кушетка/каталка]                       [сундук][вешалка]      │
│  [торшер]            — путь к югу —          [инструменты]       │
└──────────────────────────── СТЕНА ───────────────────────────────┘
```

Принципы:
- **Центр и линия разрыв↔Машина — свободны** (там идёт бой и бегут твари)
- Запад = «научная» половина (бумага/книги/доска), восток = «механическая»
  (генератор, верстак, ящики), юг = «жилой» уголок ассистента
- Крупные предметы (шкафы, генератор) — solid (коллизии), мелкие — нет

## Пол (тайловая текстура, бесшовная)

```
A seamless tileable floor texture for a 2D top-down game: Victorian
laboratory checkerboard tiles, warm cream and pale taupe squares, subtle
wear and faint scratches, thin darker grout lines, soft warm lighting.
NOT sterile white, NOT modern — 1900s scientific institution. Top-down
view, perfectly seamless/tileable, no objects, no text, 512x512.
```

## Промпты предметов (все: статичная картинка, вид сверху-3/4 как у Машины, один объект, прозрачный фон, без текста; стиль — викторианская стимпанк-лаборатория, тёплая латунь + тёмное дерево, под существующие ассеты)

Общий хвост для каждого промпта:
`Victorian steampunk laboratory style, warm brass and dark wood tones,
matches a 2D top-down RPG. One object only, centered, three-quarter
top-down view, transparent background, no text.`

1. **Лабораторный стол с колбами** (центр-запад):
```
A long laboratory workbench: dark wood top on cast-iron legs, cluttered with
glass flasks of glowing blue and green liquids, copper burners, coiled
tubing, scattered notes. {хвост}
```

2. **Книжный шкаф** (запад):
```
A tall Victorian bookcase stuffed with leather-bound tomes, rolled charts,
a few glass jars on the upper shelf, one door slightly ajar. {хвост}
```

3. **Шкаф-витрина с приборами** (север-запад):
```
A glass-front display cabinet with brass scientific instruments: sextants,
coils, vacuum tubes, small gauges, neatly arranged on velvet shelves. {хвост}
```

4. **Грифельная доска** (северо-запад):
```
A large blackboard on a wooden wheeled stand, covered in chalk formulas,
frequency diagrams and a half-erased sketch of a rift, chalk dust below. {хвост}
```

5. **Консоль управления Машиной** (восточнее Машины):
```
A steampunk control console: slanted brass panel with big levers, glowing
dials, pressure gauges and a row of blinking indicator bulbs, thick cables
running off it. {хвост}
```

6. **Катушка Теслы** (у консоли):
```
A tall Tesla coil on a wooden base: copper-wound column with a polished
toroid top, faint blue electric arcs crackling around it. {хвост}
```

7. **Генератор/динамо** (северо-восток):
```
A heavy steam dynamo generator: riveted iron body, large flywheel, copper
pipes with steam valves, a small chimney releasing a wisp of steam. {хвост}
```

8. **Верстак механика** (восток):
```
A mechanic's workbench with scattered wrenches, gears, a vice, oil cans and
a half-disassembled brass mechanism. {хвост}
```

9. **Письменный стол + кресло** (юго-запад, можно одним объектом):
```
A Victorian writing desk with an oil lamp, open journal, ink pot and quill,
and a worn leather armchair beside it. {хвост}
```

10. **Кушетка/каталка** (юг — где «отлёживаются» после переноса):
```
A simple medical cot on brass wheels with a folded wool blanket and a small
side table holding a water jug. {хвост}
```

11. **Ящики и бочка** (восток/углы, можно набором):
```
A small cluster of wooden shipping crates with stenciled markings and a
metal-banded barrel, a coil of rope on top. {хвост}
```

12. **Торшер газовый** (расставить 2-3 для света):
```
A tall Victorian gas floor lamp with a warm glowing glass globe and brass
stem on a heavy round base. {хвост}
```

## Как отдадим в игру

1. Генеришь статикой (любой размер, лучше 512×512), пуш в любую папку.
2. Я раскладываю в `public/assets/lab/props/`, регистрирую `lab_prop_*`
   и расставляю по плану кодом (крупные — с коллизией solid).
3. Дальше двигаем в игре редактором карт (` / F2), если захочешь — позиции
   экспортируются в `mapLayouts/lab.json` как в деревне.
