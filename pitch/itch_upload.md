# Веб-билд для itch.io — инструкция

## 1. Собрать билд

```bash
npm run build
```

Результат — папка `dist/` (~39 МБ). Проверено: Pitch Demo, лаборатория,
музыка работают в продакшн-сборке.

## 2. Запаковать в zip

**ВАЖНО: зипуются ФАЙЛЫ ВНУТРИ `dist/`, а не сама папка** — `index.html`
должен лежать в корне архива, иначе itch не найдёт игру.

- Linux/Mac: `cd dist && zip -r ../essence_pitch_build.zip .`
- Windows: открыть папку `dist`, выделить ВСЁ содержимое →
  ПКМ → «Отправить → Сжатая папка».

## 3. Залить на itch.io

1. itch.io → Dashboard → **Create new project**
2. **Kind of project: HTML** ← обязательно
3. Upload → выбрать zip → поставить галку **«This file will be played in the browser»**
4. **Embed options:**
   - Viewport: **1280 × 720**
   - ✅ Fullscreen button
   - ✅ Automatically start on page load (по желанию)
5. **Visibility: Draft → Restricted** — игра доступна только по секретной
   ссылке (генерируется на странице проекта). Эту ссылку и шлём издателям.
   Для публичной демки позже переключить на Public.

## 4. Что проверить после заливки (5 минут)

- [ ] Титульный экран загрузился, музыка молчит (это норм — заиграет в игре)
- [ ] Pitch Demo → пролог → деревня, музыка пошла после первого клика
- [ ] Звук/музыка работают (браузеры требуют клик — у нас обработано)
- [ ] Сейв работает (itch хранит localStorage — прогресс переживает F5)
- [ ] Открыть ссылку в приватном окне браузера — как увидит издатель

## Примечания

- `base: './'` в vite.config уже настроен — пути относительные, для itch обязательно.
- Сорсмапы выключены (вес и исходники).
- Шрифт Cormorant Garamond грузится с Google Fonts — на itch работает.
- Обновление билда: itch → Edit project → удалить старый zip, залить новый.
  Ссылка для издателей при этом НЕ меняется.
