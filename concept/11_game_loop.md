# 11. Основной игровой цикл

---

## Один цикл (5–10 минут)

```
1. Сфера появляется у камня → выбирает стартовое тело (воин / лучник / маг)
2. Идёт в мир → находит существо нового вида
3. Оценивает: выполнены ли требования параметров?
   → Тип 1: каст → захват
   → Тип 2: бой → победа → каст → захват
4. Сидит в теле → параметры растут, открываются навыки/заклинания
5. Достиг капа / хочет новое → выходит добровольно
   Или: тело убивают → Сфера вылетает
6. Ищет следующее тело → цикл повторяется
```

---

## Движущая сила — постоянный рост

Каждое новое тело — это постоянное улучшение Сферы. Нет точки где Сфера «доросла» — всегда есть следующее неизученное тело.

---

## Решения игрока в каждом цикле

| Ситуация | Вариант 1 | Вариант 2 |
|---|---|---|
| Новое тело найдено | Захватить | Остаться в текущем, запомнить место |
| Тело достигло капа | Сменить тело | Остаться: продолжать биться в нём |
| Требования не выполнены | Подобрать параметры в другом теле | Попробовать животное |
