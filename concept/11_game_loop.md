# 11. Основной игровой цикл

---

## Один цикл (5–10 минут)

```
1. Сфера появляется у камня → выбирает стартовое тело (воин / лучник / маг)
2. Идёт в мир → находит существо нового вида
3. Захватывает:
   → Passive/Fleeing: подойти → E → каст → захват живым
   → Combat: бой → победа → подойти к трупу → E → захват
4. Сидит в теле → параметры растут, учит заклинание по порогу XP
5. Достиг капа / хочет новое → ищет следующее тело
   Или: тело убивают → респаун у камня в том же теле
6. Переходит в другую зону → новые существа → цикл повторяется
```

---

## Движущая сила — постоянный рост

Каждое новое тело — новый навык + новые капы для прокачки. Нет точки где Сфера «доросла» — всегда есть следующее неизученное тело.

---

## Решения игрока в каждом цикле

| Ситуация | Вариант 1 | Вариант 2 |
|---|---|---|
| Новое тело найдено | Захватить сейчас | Остаться в текущем, вернуться позже |
| Тело достигло капа | Сменить тело | Остаться: использовать как боевой инструмент |
| Хочу другую зону | Перейти через край карты | Остаться и дофармить текущую |
| Смерть | Респаун у камня в том же теле | Можно сразу продолжить |
