# 08. Система распределения тел и лута

Правила для будущего мультиплеера.

---

## Захват тел

| Ситуация | Правило |
|---|---|
| Passive / Fleeing моб | Первый вселившийся получает тело |
| Combat моб (соло) | Больше всего урона → первым получает предложение (таймер 15 сек) |
| Босс | Кто первый начал бой — тому награда |
| В группе — Need/Greed | Несколько Need → рандом. Лидер может включить ручное распределение |
| Тело убитого игрока | Убийца не забирает тело убитого автоматически |

---

## Текущая реализация (одиночная игра)

- Passive/Fleeing → подойти + E → каст захвата → мгновенное вселение
- Combat → убить → подойти к трупу → E → каст → вселение
- Смерть тела → респаун в том же теле у камня
