# Обоз и ветераны — дизайн-спек

> Источник истины для возврата контента (ветераны + событие обоза) на карту.
> **Статус: все 7 механик реализованы** (см. ниже). Тюнинг чисел — по итогам теста.

## Состав обоза (реализовано)
- Эскорт: 1 `caravan_merchant` (лекарь, ally_heal) + **15** `caravan_guard` + 1 `scout_veteran` (защитник).
- Рейдеры (у лагеря ~5550,2350): 3 `bandit_spear` + 2 `bandit_archer` + 1 `bandit_brute` + 1 `bandit_crossbow` + по 1 `bandit_archer/brute/crossbow_veteran`.
- Фракции: эскорт = `caravan`, рейд = `raider` (фракц-война = засада).


## Ветераны — общее
- Разброс **~30 рандомных спавнов** по карте: `goblin_veteran`, `wolf_veteran`, `bear_veteran`.
- `orc_veteran` — **только один** (Вождь в орочьем лагере, через village.json), не разбрасывается.
- Каравановые (`scout_veteran`, `bandit_archer/brute/crossbow_veteran`) — **чёткие спавны** у обоза/лагеря, не разброс.
- Тело ветерана захватывается по ходу; спрайт пока — базового моба (кроме goblin_veteran, у него свой; но решили использовать обычного гоблина).
- У каждого ветерана свой квест (T2-награда), завязан на базовый квест (`prerequisiteBodyQuestId`).

## Квесты ветеранов

| Ветеран | Квест | Цель | Награда |
|---------|-------|------|---------|
| `goblin_veteran` | Ночной рейд | украсть 3 из 6-7 `orc_loot`, спавн **в орочьем лагере** (~1518,253), разброс (не у вождя) | `knife_throw` |
| `wolf_veteran` | Два удара | убить 1 `wolf_veteran` (претендент) + 2 оленя | `double_strike` |
| `bear_veteran` | Старые раны | отогнать 10 обычных волков | `mace_bash` |
| `orc_veteran` | Страж лагеря | убить 3 `goblin_veteran` (спавнятся при старте квеста, бегут на орка) | `slash_sweep` |
| `scout_veteran` | Контрмера | снять 2 `bandit_archer` (в событии обоза) | `bow_backshot` |
| `bandit_archer_veteran` | Ливень стрел | убить 4 `caravan_guard` (в обозе) | `arrow_rain` |
| `bandit_brute_veteran` | (каравановый) | убить 4 `caravan_guard` | `hammer_smash` |
| `bandit_crossbow_veteran` | (каравановый) | убить 4 `caravan_guard` | `crossbow_snare` |

Примечание: `orc_veteran` = «Вождь орков» (цель `bq_orc`). Обычные орки **не агрятся** на него (чиф-респект).

## Событие «Обоз»

**Цикл (односторонний):**
- A = **Вальдмар** (8800,2800) — отправление (НЕ старт игрока Эшворт).
- B = **Эшворт** (3200,2800) — назначение.
- Маршрут по дороге A→B, мимо райдерского лагеря (~5550,2350).
- Спавн в A: обоз + эскорт + рейдеры. Стоит **120с** (окно вселения в эскорт — только в A).
- Выдвигается → в радиусе **~700px** рейдеры «просыпаются» → атака.
- Доезжает до B (или уничтожен) → стоит **30с** → **ВСЁ исчезает** (обоз+эскорт+рейдеры) → респаун в A **полным составом**. **Тело игрока не трогаем.**

**Состав (полный, респаунится целиком каждый цикл):**
- Эскорт: 1 `caravan_merchant` + 5 `caravan_guard`.
- Рейдеры: 3 `bandit_spear` + 2 `bandit_archer` + 1 `bandit_brute` + 1 `bandit_crossbow`.

**Правила:**
- Рейдеры сначала добивают **весь эскорт**, потом телегу (повозка неуязвима, пока жив хоть один охранник).
- Рейдеры у лагеря **пассивны** и привязаны к лагерю (не агрессивны, далеко не уходят) — чтобы игрок успел вселиться.
- Вселиться в эскорт — только в точке A на стоянке.
- A и B — сейф-зоны городов (рейдеры в город не лезут; доехал = спасся).

## Механики для реализации (по порядку)
1. ~30 рандом-спавнов на каждый тип ветерана.
2. Якорь спавна квест-объектов на координаты (orc_loot в лагере).
3. Спавн врагов при старте квеста (goblin_veteran на orc_veteran).
4. Фракц-респект (свои/вождь не агрятся на вселённое тело).
5. Обоз-цикл (A→B, тайминги, despawn-all-кроме-игрока, полный респаун).
6. Пассивный отряд у лагеря + триггер атаки по близости обоза.
7. Рейдеры: эскорт → потом телега.
