# 21. Глава 1 — Структура карты и мобы

## Размер карты

60 × 40 тайлов · тайл = 32px · итого 1920 × 1280 пикселей

## Зоны на карте

Центр карты — деревня Эшворт (тайл 30,20 = пиксель 960,640).
Четыре зоны расходятся в стороны света.

```
        СЕВЕР
   [Туманное озеро 🌊]
    тайлы x:15-45 y:2-14

[Каменные    [ЭШВОРТ]    [Пепельная
 холмы 🪨]   x:24-36      роща 🔥]
 x:2-16      y:16-24      x:44-58
 y:14-30                  y:14-30

        [Вершины ветров 💨]
         тайлы x:15-45 y:28-38
        ЮГ
```

### Зона 1 — Пепельная роща (Огонь) 🔥
- **Тайлы:** x: 44–58, y: 14–30
- **Пиксели:** x: 1408–1856, y: 448–960
- **Тайл поверхности:** `tile_stone` (обожжённая земля, серо-оранжевый тинт)
- **Мобы:** Искра, Пепельник
- **Босс:** Игнис — центр зоны (тайл 51,22 = пиксель 1632,704)
- **Особенности:** горячие источники (декоративные), горящие деревья

### Зона 2 — Туманное озеро (Вода) 🌊
- **Тайлы:** x: 15–45, y: 2–14
- **Пиксели:** x: 480–1440, y: 64–448
- **Тайл поверхности:** `tile_grass` (синеватый тинт у воды)
- **Мобы:** Брызгун, Туманник
- **Босс:** Акварис — центр зоны (тайл 30,8 = пиксель 960,256)
- **Особенности:** водная арена (остров), туман снижает видимость

### Зона 3 — Каменные холмы (Земля) 🪨
- **Тайлы:** x: 2–16, y: 14–30
- **Пиксели:** x: 64–512, y: 448–960
- **Тайл поверхности:** `tile_stone`
- **Мобы:** Каменыш, Грязевик
- **Босс:** Терра — глубина пещеры (тайл 8,22 = пиксель 256,704)
- **Особенности:** узкие проходы между скалами, пещерный зал босса

### Зона 4 — Вершины ветров (Ветер) 💨
- **Тайлы:** x: 15–45, y: 28–38
- **Пиксели:** x: 480–1440, y: 896–1216
- **Тайл поверхности:** `tile_stone` (скала утёса)
- **Мобы:** Вихрик, Свистун
- **Босс:** Аэрос — вершина утёса (тайл 30,35 = пиксель 960,1120)
- **Особенности:** открытая площадка, края = мгновенная смерть

### Центр — Деревня Эшворт 🏘️
- **Тайлы:** x: 24–36, y: 16–24
- **Пиксели:** x: 768–1152, y: 512–768
- **Тайл поверхности:** `tile_grass`
- **Здания:** таверна, кузница, рынок, дома (декор)
- **NPC:** Мира, Берт, Aldric, Пол, Странник

## Реализация в коде

### Новые поля BodyDefinition
```ts
element?: 'fire' | 'water' | 'earth' | 'wind';  // стихия — для слабостей
isBoss?: boolean;                                  // босс — увеличенный спрайт, особый HP-бар
spawnZone?: 'fire' | 'water' | 'earth' | 'wind' | 'village';
```

### Порядок реализации
1. Добавить 8 новых `BodyDefinition` в `src/data/bodies.ts`
2. Добавить текстуры мобов в `BootScene.genCharTextures()`
3. Создать `src/data/chapter1.ts` — точки спавна зон, конфиг карты
4. В `GameScene`: разбить карту на зоны тайлами (тинт/тип тайла по зоне)
5. Спавнить мобов из `chapter1.ts` вместо текущего захардкоженного спавна
6. Боссов — отдельная сущность `Boss extends Creature` с фазами
