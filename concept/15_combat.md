# 15. Механика боя — формулы

---

## Физический урон

### Шаг 1 — Проверка попадания
```
Шанс попасть = Точность / (Точность + Уклонение цели)
```
При равных статах — 50% шанс попасть.

### Шаг 2 — Базовый урон
```
Ближний бой:  База = 5 + Сила × 2
Дальний бой: База = 5 + Ловкость × 2
```

### Шаг 3 — Снижение Стойкостью
```
Снижение = Стойкость / (Стойкость + 125)
Итоговый урон = База × (1 − Снижение)
```
Максимум 80% при Стойкости 500.  
Стартовый воин (Стойкость 25) → 17% снижения.

### Шаг 4 — Критический удар
```
Шанс крита = Удача / (Удача + 50)
Крит = Урон × 1.5
```

### Примеры

| Сила | Стойкость цели | База | Снижение | Итого |
|---|---|---|---|---|
| 10 | 25 | 25 | 17% | ~21 |
| 10 | 100 | 25 | 44% | ~14 |
| 10 | 200 | 25 | 62% | ~9 |
| 30 | 25 | 65 | 17% | ~54 |
| 30 | 100 | 65 | 44% | ~36 |

## Магический урон

```
База = 3 + Интеллект × 2.5
Снижение = Воля / (Воля + 125)
Итоговый урон = База × (1 − Снижение)
```
Максимум 80% при Воле 500.  
Магия сильнее физики (2.5 vs 2.0), но требует Ману.  
Проверка попадания для магии — **нет** (от магии нельзя уклониться).

| Воля | Магзащита |
|---|---|
| 15 (стартовый маг) | 11% |
| 50 | 29% |
| 100 | 44% |
| 200 | 62% |
| 500 (макс) | 80% |

## Максимальные HP и Мана

```
Макс HP   = 50 + Здоровье × 5
Макс Мана = 20 + Мана-стат × 3
```

## Регенерация — система GW1

Без зелий и бутылок. Игрок управляет ресурсами через умения и статусы.

```
Реген HP   = Живучесть × 0.15 HP/сек  (всегда)
Реген Маны = Мана-стат × 0.2  Маны/сек (всегда)
```

### Статусы влияющие на реген

| Статус | Источник | Эффект |
|---|---|---|
| Замедление регена Маны | Умение/заклинание | −50% реген Маны |
| Блокировка регена Маны | Умение/заклинание | 0 реген Маны |
| Усиление регена Маны | Умение/заклинание | +100% реген Маны (временно) |
| Кровотечение | Оружие | блокирует реген HP |
| Отравление | Оружие | HP убывает +доп урон в сек |

### Почему это подходит игре
- Без зелий — меньше систем, чище геймплей
- Тело с высокой Живучестью/Маной — реальное преимущество
- Статусы становятся ключевыми в PvP
- Побеждает скилл, а не количество зелий

## Скорость атаки (Кулдаун)

```
Кулдаун атаки (сек) = max(0.5, 2 − Скорость × 0.03)
```

| Скорость | Кулдаун |
|---|---|
| 1 | 1.97 сек |
| 20 | 1.4 сек |
| 40 | 0.8 сек |
| 50+ | 0.5 сек (минимум) |
