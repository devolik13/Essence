# 15. Механика боя — формулы

---

## Физический урон

### Шаг 1 — Проверка попадания
```
Шанс попасть = Точность / (Точность + Уклонение цели)
```
При равных статах — 50% шанс попасть.

### Шаг 2 — Базовый урон
```
Ближний бой:  База = 5 + Сила × 2
Дальний бой: База = 5 + Ловкость × 2
```

### Шаг 3 — Снижение Стойкостью
```
Снижение = Стойкость / (Стойкость + 125)
Итоговый урон = База × (1 − Снижение)
```
Максимум 80% при Стойкости 500.  
Стартовый воин (Стойкость 25) → 17% снижения.

### Шаг 4 — Критический удар
```
Шанс крита = Удача / (Удача + 50)
Крит = Урон × 1.5
```

### Примеры

| Сила | Стойкость цели | База | Снижение | Итого |
|---|---|---|---|---|
| 10 | 25 | 25 | 17% | ~21 |
| 10 | 100 | 25 | 44% | ~14 |
| 10 | 200 | 25 | 62% | ~9 |
| 30 | 25 | 65 | 17% | ~54 |
| 30 | 100 | 65 | 44% | ~36 |

## Магический урон

```
База = 3 + Интеллект × 2.5
Снижение = Воля / (Воля + 125)
Итоговый урон = База × (1 − Снижение)
```
Максимум 80% при Воле 500.  
Магия сильнее физики (2.5 vs 2.0), но требует Ману.  
Проверка попадания для магии — **нет** (от магии нельзя уклониться).

| Воля | Магзащита |
|---|---|
| 15 (стартовый маг) | 11% |
| 50 | 29% |
| 100 | 44% |
| 200 | 62% |
| 500 (макс) | 80% |

## Максимальные HP и Мана

```
Макс HP   = 50 + Здоровье × 5
Макс Мана = 20 + Мана-стат × 3
```

## Регенерация — система GW1

Без зелий и бутылок. Игрок управляет ресурсами через умения и статусы.

```
Реген HP   = Живучесть × 0.1 HP/сек  (всегда)
Реген Маны = Мана-стат × 0.1  Маны/сек (всегда)
```

| Живучесть / Мана | Реген/сек | Время заполнения (с нуля) |
|---|---|---|
| 1 (старт) | 0.1 | ~550 сек (HP) / ~200 сек (Мана) |
| 15 (стартовый маг) | 1.5 | ~43 сек (Мана 65) |
| 100 | 10 | ~255 сек (HP) / ~152 сек (Мана) |
| 500 (макс) | 50 | ~51 сек (HP) / ~30 сек (Мана) |

> Коэффициент 0.1 — предварительный. Уточняется после определения стоимости заклинаний.

### Статусы влияющие на реген

| Статус | Источник | Эффект |
|---|---|---|
| Замедление регена Маны | Умение/заклинание | −50% реген Маны |
| Блокировка регена Маны | Умение/заклинание | 0 реген Маны |
| Усиление регена Маны | Умение/заклинание | +100% реген Маны (временно) |
| Кровотечение | Оружие | блокирует реген HP |
| Отравление | Оружие | HP убывает + доп урон в сек |

## Кулдаун атаки — фиксированный по типу оружия

Скорость убрана из статов. Кулдаун определяется типом оружия/действия.

| Оружие / Действие | Кулдаун |
|---|---|
| Кинжал | 0.8 сек |
| Меч | 1.2 сек |
| Двуручник | 2.0 сек |
| Копьё | 1.8 сек |
| Булава | 1.3 сек |
| Молот | 2.2 сек |
| Лук короткий | 1.0 сек |
| Лук длинный | 1.8 сек |
| Арбалет | 2.5 сек |
| Заклинание (базовое) | зависит от заклинания |

> Скорость передвижения — базовая у всех, меняется умениями и статусами (Рывок, Замедление и т.д.)

## Регенерация — процентная система

Время заполнения одинаковое при любом запасе HP/Маны.

```
Реген HP/сек   = Макс HP ÷ 60   (полный бар за 60 сек)
Реген Маны/сек = Макс Мана ÷ 30  (полный бар за 30 сек)
```

| | HP (Здоровье 500) | Мана (Мана 500) |
|---|---|---|
| Макс | 2 050 | 100 |
| Реген/сек | 34.2 | 3.3 |
| Время заполнения | **60 сек** | **30 сек** |
