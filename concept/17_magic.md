# 17. Система магии

> Адаптация из пошаговой тактики Archimage под реалтайм MMORPG.  
> Оригинал: `wip/archimage_spells.md`

---

## Общие принципы магии

- Заклинания требуют **Интеллект** (урон) и **Мана** (ресурс)
- Каждое заклинание имеет: **время каста**, **кулдаун**, **стоимость маны**
- Пассивные заклинания («ауры») — активны пока есть мана, расходуют X маны/сек
- Призывы — временные тела без Сферы, живут пока не убиты или не закончится мана
- Канальные заклинания — прерываются при получении урона или Сбитии концентрации

### Адаптация пошаговых механик

| Было (пошаговое) | Стало (реалтайм) |
|---|---|
| 1 ход | ~2 сек |
| «Пассивка весь бой» | Аура: включается, расходует ману/сек |
| «Весь ряд» | Линия 8м |
| «АОЕ 3×3» | Радиус 4м |
| «Вся территория врага» | Радиус 12м |
| «По колонкам» | Конус 90° |
| «Прерывание X%» | X% шанс наложить Сбитие концентрации |
| Стаки брони | Стойкость (физ.) / Воля (маг.) |

## Привязка школ к архетипам

Школа магии изучается через тело соответствующего архетипа.  
Гибридные тела могут дать доступ к двум школам.

| Школа | Основной архетип | Тип заклинаний |
|---|---|---|
| Огонь | Маг | Урон, АОЕ, зоны |
| Вода | Чародей | Контроль, замедление |
| Ветер | Ассасин | Крит, скорость, линейный урон |
| Земля | Рыцарь | Защита, броня, призывы-стены |
| Природа | Жрец | Хил, призывы, реген |
| Яд | Ассасин / Алхимик | DoT, зоны яда |
| Свет | Паладин | Хил, анти-призывы, дебафф-снятие |
| Тьма | Вампир | Дебаффы, иссушение, % урон |
| Некромантия | Некромант | Призывы нежити, щиты |
| Кровь | Вампир | Лайфстил, жертва, отражение |
