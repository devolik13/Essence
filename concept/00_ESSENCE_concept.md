## 1. Концептуальное ядро

**Жанр:** PvP MMORPG  
**Рабочее название:** Сущность (Сфера)

Игрок — бестелесная Сущность (Сфера), которая вселяется в существ. Тело — ресурс, не аватар. Смерть тела не конец игры — Сфера вылетает и ищет новое тело.

### Два состояния игрока
- **Астральный режим** (вне тела) — видит других Сфер, ищет носителя. Переход не мгновенный — окно уязвимости.
- **Физический режим** (внутри тела) — сражается, качается, учит навыки и заклинания.

### Камень возрождения
При смерти тела Сфера появляется у камня возрождения. Там доступны бесплатные базовые сосуды на выбор: воин, лучник, маг.

### Визуальная идентичность
- Игрок выбирает режим отображения: реальное тело или внешность Сферы поверх него.
- Сфера кастомизируется независимо от носителя (цвет, форма, аура, эффекты).

## 2. Параметры и прогрессия

Все 12 параметров стартуют со значения 1 и принадлежат Сущности навсегда.

| Параметр | Описание |
|---|---|
| Сила | Урон в ближнем бою |
| Ловкость | Урон в дальнем бою |
| Точность | Шанс попасть по цели |
| Уклонение | Шанс избежать удара |
| Скорость | Передвижение и скорость действий |
| Здоровье | Максимальный запас HP |
| Живучесть | Выживаемость и регенерация |
| Стойкость | Снижение физического урона (ближний + дальний) |
| Интеллект | Магический урон (сила заклинаний растёт с Интеллектом) |
| Воля | Магическая защита |
| Мана | Запас магической энергии |
| Удача | Крит и случайные события |

### Ранг Сущности
Ранг = Сумма всех 12 параметров ÷ 10. Растёт автоматически. Используется как мерило прогресса и требование для захвата редких тел.

### Боевые формулы
- Попадание = Точность / (Точность + Уклонение противника)
- Физический урон (Сила и Ловкость) снижается Стойкостью
- Магический урон (Интеллект) снижается Волей

### Прогрессия параметров — гибрид
- Опыт копится от действий и автоматически уходит в нужный параметр тела
- Игрок выбирает режим: все параметры / один / два
- Чем выше параметр — тем дороже следующий уровень
- При достижении капа опыт перетекает в остальные параметры

### Пороги опыта открывают навыки
Параллельно росту параметров накопленный опыт в теле открывает навыки и заклинания по порогам.  
Пример: паладин 10 000 опыта = навык, 20 000 опыта = заклинание.

## 3. Система тел

### Два типа тел
- **Тип 1 — пассивные:** всегда готовы к захвату (заяц, птица и т.д.)
- **Тип 2 — боевые:** сначала победи, потом захвати
- Животные — только природные атаки в бою, без оружия и магии

### Тела с несколькими основными параметрами
Пример — паладин: Сила кап 50, Живучесть кап 70, Воля кап 40. Игрок выбирает приоритет распределения опыта между параметрами.

### Система капов
- Каждое тело имеет потолок роста по каждому параметру
- Параметр не падает при смене тела — просто не растёт выше капа нового носителя
- Навыки и заклинания остаются у Сущности навсегда

### Коллекция тел
- Запасных тел при себе нет
- Дом — безлимитное хранилище, но один экземпляр каждого вида
- Нельзя запастись 10 паладинами — только один

### Прокачка только через захват и бои
Никаких гильдий и учителей. Навыки, заклинания и параметры — только через тела существ и убийства боссов. Условие освоения навыка — параметры должны соответствовать требованиям.

## 5. Оружие

### Ближний бой — 6 видов (3 типа × 2 стиля)

| Оружие | Тип урона | Стиль | Свойство | Умение 1 | Умение 2 |
|---|---|---|---|---|---|
| Кинжал | Колющий | Быстрый | Отравление | Укол | Бросок кинжала (возвращается) |
| Копьё | Колющий | Медленный | Отбрасывание | Выпад (дистанция) | Удар древком (вплотную, отброс) |
| Меч | Режущий | Быстрый | Замедление движения | Удар | Двойной удар (два быстрых) |
| Двуручник | Режущий | Медленный | Кровотечение | Рубящий удар | Размах (конус перед собой) |
| Булава | Дробящий | Быстрый | Сбитие концентрации | Удар | Удар плашмя (повышенный шанс) |
| Молот | Дробящий | Медленный | Пробитие брони | Удар | Сильный удар (по земле, AOE) |

### Дальний бой — 3 вида

| Оружие | Свойство | Умение 1 | Умение 2 |
|---|---|---|---|
| Короткий лук | Метка (цель получает больше урона) | Выстрел (короткая дистанция) | Выстрел с отскоком назад |
| Длинный лук | Быстрая перезарядка (шанс сбросить кулдаун) | Выстрел (дальняя дистанция) | Дождь стрел (AOE) |
| Арбалет | Пробитие (снаряд летит сквозь до 3 целей) | Выстрел (средняя дистанция) | Пробивающий болт (3 цели в ряд) |

### Типы умений
- **Привязанные к оружию** — работают только с нужным оружием в руках
- **Общие** — работают с любым оружием, привязаны к архетипу
- **Магические** — требуют Ману и Интеллект

## 6. Астральная броня и профессии

### Астральная броня
- Принадлежит Сущности навсегда
- Даёт постоянные бонусы в любом теле
- Защищает Сферу вне тела
- Куётся Астральным кузнецом

### 7 профессий — качаются в любом теле

| Профессия | Чем занимается |
|---|---|
| Астральный кузнец | Кует астральную броню для Сфер |
| Кузнец | Кует оружие и снаряжение |
| Фермер | Выращивает ресурсы (овощи, фрукты) |
| Повар | Готовит усилители и еду |
| Алхимик | Варит снадобья и зелья |
| Травник | Собирает травы и растения |
| Рудокоп | Добывает руду и камень |

## 8. Система распределения тел и лута

| Ситуация | Правило |
|---|---|
| Тип 1 (пассивный моб) | Первый вселившийся получает тело |
| Тип 2 вне группы | Больше всего урона нанёс — первым получает предложение (таймер 15 сек) |
| Босс | Кто первый начал бой — тому награда |
| В группе — Need/Greed | Несколько Need → рандом. Лидер может включить ручное распределение |
| Тело убитого игрока | Убийца не забирает тело убитого автоматически |
