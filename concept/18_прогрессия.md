# 18. Прогрессия: опыт, статы, умения

---

## Как Сфера получает статы от тела

Пока Сфера находится внутри тела, она получает **очки опыта (XP)** за боевые действия.  
XP автоматически уходит в параметры тела — но только до их **капа**.

### Источники XP

| Действие | Растёт параметр |
|---|---|
| Нанесён удар в ближнем бою | Сила |
| Нанесён удар в дальнем бою | Ловкость, Точность |
| Применено заклинание | Интеллект, Мана |
| Получен удар (устоял) | Стойкость, Здоровье |
| Выжил при магическом уроне | Воля |
| Уклонение сработало | Уклонение |
| Убит враг | Все основные параметры тела |

### Режимы распределения XP

Игрок выбирает в настройках:
- **Авто** — XP равномерно идёт в основные параметры тела
- **Фокус** — XP идёт в один выбранный параметр (остальные не растут)

---

## Стоимость роста параметра

```
XP на следующий уровень = текущее_значение × 10
```

Пример: поднять параметр с 5 до 6 стоит 50 XP, с 20 до 21 — 200 XP.  
Чем выше кап тела — тем дольше прокачка.

---

## Получение умения

**Умение открывается когда ПЕРВЫЙ основной параметр тела достигает своего капа.**

- Умение навсегда остаётся у Сферы
- Второй захват того же типа тела — ничему не учит (умение уже есть)
- Если Сфера покинула тело до достижения капа — умение не получено

---

## Уровень игрока

Отдельного «уровня» нет. Мерило прогресса — **Ранг Сферы**:

```
Ранг = сумма всех 12 параметров ÷ 10
```

Ранг растёт автоматически по мере накопления статов из тел.  
Используется как требование для захвата редких и легендарных существ.

---

## Итоговый цикл

```
Войти в тело → Драться → XP → Параметр растёт → Достиг капа → Умение получено → Выйти → Найти следующее тело
```
