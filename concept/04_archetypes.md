# 04. Архетипы существ

Каждое существо имеет капы по 1-3 параметрам, определяющие его "архетип". Существа группируются по основному стату.

---

## Группы по основному стату

### Сила (ближний бой)
| Оружие | Существа |
|--------|---------|
| Меч | Волк, Волк-вожак |
| Булава | Медведь, Медведь-старейшина |
| Двуручник | Орк, Орк-военачальник |
| Копьё | Копейщик, Копейщик-ветеран |
| Молот | Громила, Громила-берсерк |

### Ловкость (ближний + дальний)
| Оружие | Существа |
|--------|---------|
| Кинжал | Гоблин, Гоблин-ветеран |
| Кастеты | (будущие существа) |
| Кор. лук | Разведчик, Капитан разведчиков |
| Дл. лук | Разбойник-лучник, Снайпер |
| Арбалет | Арбалетчик, Арбалетчик-мастер |

### Интеллект (магия)
| Посох | Существа |
|-------|---------|
| Огненный | Искра, Пепельник |
| Ледяной | Брызгун, Туманник |
| Каменный | Камешек, Грязевик |
| Штормовой | Вихрик, Свистун |
| Природы | Волк-дух, Шаман |

### Особые
| Тип | Существа |
|-----|---------|
| Пассивные | Кролик (Passive), Спирит (Fleeing) |
| Боссы | Игнис, Акварис, Терра, Аэрос |

---

## Прогрессия по зонам

| Зона | Существа | Учат |
|------|---------|------|
| Деревня | rabbit, spirit, goblin, wolf, bear, orc, scout | T1 навыки |
| Стихийные | элементали, ветераны, бандиты | T1 стихий + T2 оружия |
| Боссы | Игнис, Акварис, Терра, Аэрос | Зачарования |

---

## Комбинированные архетипы (будущее)

Существа с 2-3 основными параметрами:
- Паладин: Сила + Стойкость + Воля
- Рейнджер: Ловкость + Точность + Уклонение
- Боевой маг: Интеллект + Стойкость
